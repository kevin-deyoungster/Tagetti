# Tagetti
A 2D skill-based arcade game I built with friends at the Global Game Jam, Ashesi

## Play 
[Play on the Google Playstore](https://play.google.com/store/apps/details?id=com.ggjAshesi.tagetti&hl=en)

[Play on CrazyGames](https://crazygames.com/game/tagetti)

Built for [Global Game Jam 2017](https://github.com/kevin-deyoungster/Tagetti-Backend-API)
